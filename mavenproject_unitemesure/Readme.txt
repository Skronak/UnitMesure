Projet 2013-2014, MIAGE NANTES

Réalisé dans le cadre des cours de developpement logiciel et de NPIE.
Réalisé en JAVA avec NetBeans,le plugin MAVEN et JUNIT.

Bibliothèque de fonction destinée à etre utilisée afin d'implémenter un système de conversions d'unités de mesures.


Le package (unite_mesureUnitMesure / mavenproject_unitemesure / src / main / java / com / unite_mesure /) contient 
l'ensemble des classes de la bibliothèque.
 	AbstractConversion.java : Classe abstraite conversion 	
	Addition.java 	        : Classe fille de Conversion
	Complexe.java  	        : Classe fille de Conversion	
	GestionConversion.java  : Classe gérant les opération de conversion d'unités de mesures
	Multiplication.java  	  : Classe fille de Conversion
	Quantite.java           : Classe indiquant une quantité d'unités
	TypeUnite.java          : Enumération des types d'unités de mesures
	Unite.java              : Classe définissant une Unité du programme
	Validator.java          : Classe de validation des saisies utilisateurs
